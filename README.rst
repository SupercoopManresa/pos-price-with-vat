==================
POS Price with VAT
==================

Odoo module to show prices with VAT on the POS product's list

Dependencies
============

- [Beesdoo Product module](https://github.com/beescoop/Obeesdoo/tree/12.0/beesdoo_product): Reads the field `total_with_vat`
